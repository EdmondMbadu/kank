Critères pour être la meilleur employé :

  • Pourcentage le plus élevé (avec equipe de 60+ personnes)

Critères pour être le meilleur Manager :

  • Pourcentage le plus élevé de Manager
  • Nombre de personnes nouvellement inscrites du 1er au 30 (ex : de 60 à 89, soit 29 personnes).
  • Est-ce que votre équipe a été la meilleure ?

    • Le manager qui remplit 2 des 3 critères gagne.

Critères pour être la meilleure Equipe :

  • Avoir une réserve avec le plus d’argent.

Informations générales sur le prêt

  • Le budget pour prêter de l'argent sera déterminé sous la Trésorerie-> le Budget Emprunts Du Mois. Suivez ce montant de près, si le montant atteint 0, vous pouvez demander plus, mais vous ne pouvez pas accepter de nouveaux emprunteurs jusqu'à ce que des fonds supplémentaires soient alloués à votre budget.
